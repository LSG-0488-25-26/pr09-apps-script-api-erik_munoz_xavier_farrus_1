# pr09-apps-script-api

Este proyecto es una aplicación Android desarrollada por Xavier Farrús y Erik Muñoz. La app consume una API creada a partir de un documento Excel mediante Google Apps Script (JavaScript), permitiendo gestionar usuarios, productos y favoritos.

## Descripción del Proyecto

- **Origen de la API:**
  - La API se genera desde un archivo Excel en Google Sheets usando un script de Google Apps Script (JavaScript).
  - El script convierte los datos del Excel en endpoints accesibles desde la app.
- **Autores:** Xavier Farrús y Erik Muñoz
