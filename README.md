# pr09-apps-script-api

Este proyecto es una aplicación Android desarrollada por Xavier Farrús y Erik Muñoz. La app consume una API creada a partir de un documento Excel mediante Google Apps Script (JavaScript), permitiendo gestionar usuarios, productos y favoritos.

## Descripción del Proyecto

- **Origen de la API:**
  - La API se genera desde un archivo Excel en Google Sheets usando un script de Google Apps Script (JavaScript).
  - El script convierte los datos del Excel en endpoints accesibles desde la app.
- **Autores:** Xavier Farrús y Erik Muñoz

## Recursos

- **Documento Excel:** [Enlace al Excel](https://docs.google.com/spreadsheets/d/1zv7ESq1NR2FJOQa-R_0cZ3MzKBs7MRuoalvW3txv6qg/edit?usp=sharing)
- **Script JS:** Se puede acceder desde extensiones, Apps Scripts y ahi será visible el codigo. Todo esto atraves del link del excel de arriba.

## Cómo funciona

1. El usuario se registra o inicia sesión.
2. Puede ver el listado de productos, crear nuevos y marcar favoritos.
3. Todos los datos se gestionan a través de la API conectada al Excel.
